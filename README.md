# 🐾 PetShop — Sistema de Gestão

Sistema completo para gerenciamento de pet shop desenvolvido com Java Spring Boot e React.

## 🚀 Tecnologias

### Backend
- Java 17
- Spring Boot 3
- Spring Security + JWT
- PostgreSQL
- JPA/Hibernate

### Frontend
- React
- Bootstrap
- Html

## 📋 Funcionalidades

- ✅ Autenticação com JWT
- ✅ Cadastro de Clientes
- ✅ Cadastro de Pets
- ✅ Agendamentos de serviços
- ✅ Controle de Produtos e Estoque
- ✅ Registro de Vendas
- ✅ Cadastro de Funcionários
- ✅ Dashboard com métricas

## ⚙️ Como rodar

### Backend
```bash
# Configure o banco no application.properties
# Execute pelo IntelliJ ou:
./mvnw spring-boot:run
```

### Frontend
```bash
cd petshop-front
npm install
npm start
```

## 🗄️ Banco de Dados

PostgreSQL — tabelas: usuarios, clientes, pets, agendamentos, produtos, vendas, itens_venda, funcionarios
